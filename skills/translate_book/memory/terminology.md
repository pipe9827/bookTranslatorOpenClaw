# Terminology Glossary

This file maintains consistent Spanish-to-English term mappings used across all
translations. The pipeline reads this file before each translation pass and
updates it automatically when new bilingual glosses are detected.

## Format

Each row follows the pattern:

```
| Spanish Term | English Term | Notes |
```
